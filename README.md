# Gesture-Controlled-Inverting-Filter
Fun personal project that allows for live visual effects for creative events, video fx, projection mapping etc. 

This Python script utilizes OPENCV and MEDIAPIPE to track user's hand gesture and inverts color when "OPEN HAND" is detected. 
# BE SURE TO DOWNLOAD THESE LIBRARIES:
-import cv2

-import mediapipe 

-import numpy


To start front-facing monitor, simply run the script. To exit script, press 'q' (case insensitive).

Project renderer: Apple M1 Pro (MAY NOT BE SUITABLE FOR ALL MACOS)
